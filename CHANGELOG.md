# Changelog

All notable changes to **Granola_bar** will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/).

## [0.1.0] — 2026-04-23

### Added

- System audio capture via macOS ScreenCaptureKit
- Local Whisper transcription (on-device, no cloud)
- Gemini 2.5 Flash note enhancement with transcript hyperlinks
- TipTap rich text editor with auto-save
- Meeting templates (General, 1-on-1, Sales, Interview, Pitch)
- Semantic vector search across all meetings
- Meeting library with folder organization
